Manual tests for setting window.opener to null

This resource contains various tests; when in doubt, start a fresh session for each

  1. Click "same-origin", then "example.com". Observe that there's a single opened window (besides this one) navigated to "example.com".
  2. Click "same-origin", then the link in the opened window, then "example.com". Observe that there's two opened windows (besides this one) one navigated to "example.com" and to "example.org".
  3. Click "cross-origin", then "example.com". Observe that there's two opened windows (besides this one).
